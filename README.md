# Multi-Flow: Multi-View-Enriched Normalizing Flows for Industrial Anomaly Detection

This is the official code to the paper 
[**"Multi-Flow: Multi-View-Enriched Normalizing Flows for Industrial Anomaly Detection"**](https://arxiv.org/abs/2504.03306) 
by Mathis Kruse and Bodo Rosenhahn, which was accepted to the CVPR 2025 Workshop VAND 3.0.

The code will be published in conjunction with CVPR 2025 around June/July.
